# Gesture Recognition Module

## Supported Gestures

| Class         | Description        |
|---------------|--------------------|
| `grab`        | Grab / clench      |
| `swipe_up`    | Swipe up           |
| `swipe_down`  | Swipe / swipe down |
| `swipe_left`  | Swipe left         |
| `swipe_right` | Swipe right        |
| `finger_heart` | Finger heart       |
| `wave`        | Wave               |
| `noise`       | Idle / noise       |

```bash
pip install -r requirements.txt
````

## Usage

### Train

```bash
python train.py
```

### Inference (webcam)

```bash
python inference.py
```
